# Contributing to Syn'Tek

We welcome all "Primes" (contributors) to the network.

## Rules for New Words
To add a word to Syn'Tek, it must pass the **"AI Ambiguity Test"**.

1.  **No Homonyms:** The word must have only one meaning in a technical context.
    * *Bad:* "Date" (Fruit? Calendar? Romance?)
    * *Good:* "SESH" (Session/Meeting) or "CHRONO" (Time).
2.  **Universal:** Must be understood by standard LLMs (GPT/Claude).
3.  **Style:** Must sound Tech, Military, or Sci-Fi.

## How to Submit
1.  Fork the repo.
2.  Add your token to `DICTIONARY.md`.
3.  Submit a Pull Request with an example sentence.
